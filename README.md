### Amazon Website Clone

This is a **responsive Amazon homepage clone** built using **HTML, CSS, and JavaScript**. The project replicates the basic layout and functionality of the Amazon website, focusing on frontend design, interactive UI elements, and user experience.

Features

* Amazon-style **navigation bar**
* **Search bar with category filtering**
* **Product grid layout**
* **Add to Cart functionality**
* **Cart sidebar with item management**
* **Hero image slider**
* **Toast notifications**
* **Footer section**
* **Responsive design using CSS Flexbox**

Technologies Used

* HTML5 – Structure of the website
* CSS3 – Styling and layout design
* JavaScript (Vanilla JS) – Interactive features and functionality
  
What I Learned

* DOM manipulation using JavaScript
* Building interactive UI components
* Managing cart data using **localStorage**
* Structuring responsive layouts using **Flexbox**

Purpose
This project was built to practice frontend development and improve layout structuring skills.
